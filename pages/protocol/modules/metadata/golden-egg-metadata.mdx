# GoldenEggMetadata

**[`contracts/modules/GoldenEggMetadata.sol`](https://github.com/soundxyz/sound-protocol/blob/master/contracts/modules/GoldenEggMetadata.sol)**

A metadata module that implements the sound.xyz Golden Egg, a lottery mechanism that rewards a unique 1-of-1 NFT per edition.

Inherits:

- [IMetadataModule](/protocol/core/i-metadata-module)

### getGoldenEggTokenId

```solidity
function getGoldenEggTokenId(ISoundEditionV1 edition) external view returns (uint256)
```

Returns token ID for the golden egg after the [`SoundEdition.mintRandomness`](/protocol/core/sound-edition#mintrandomness) is locked, else returns 0.

| Params:   |                                                          |
| --------- | -------------------------------------------------------- |
| `edition` | Address of the song edition contract we are minting for. |
